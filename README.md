# Challenge Hub

Веб-приложение для челленджей между друзьями: регистрация пользователей, начисление очков и рейтинг участников.

## Возможности
- Регистрация и вход.
- Каталог челленджей и отметка выполненных.
- Автоматический подсчет очков.
- Рейтинг участников.

## Технологии
- Python 3.10+
- Flask
- SQLite (через SQLAlchemy)

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Откройте браузер: `http://127.0.0.1:5000`.
